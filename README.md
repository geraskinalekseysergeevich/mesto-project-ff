# Проектная работа Mesto

Проект **Mesto** — это интерактивная галерея мест, где пользователи могут создавать карточки с изображениями, ставить лайки, удалять их и редактировать свой профиль. Также реализована возможность обновления аватара.  

Проект выполнен в рамках обучения и демонстрирует работу с **REST API**, валидацию форм, модальные окна и современный UX.

## Ссылка

https://geraskinalekseysergeevich.github.io/mesto-project-ff/

---

## Функционал

* Редактирование профиля (имя и описание)
* Добавление новых карточек с изображениями
* Удаление карточек с подтверждением
* Лайки на карточки (поставить/снять)
* Просмотр изображений в увеличенном виде
* Обновление аватара с валидацией URL
* Валидация всех форм (обязательные поля, корректный формат)
* UX: индикатор загрузки при сохранении данных

---

## Технологии

* **HTML5, CSS3, JavaScript (ES6+)**
* Работа с DOM и событиями
* REST API (методы GET, POST, PATCH, DELETE)
* Асинхронные запросы через `fetch`
* Принципы модульности: разделение логики на `card.js`, `api.js`, `modal.js`, `validation.js`, `index.js`, `ux.js`
* Адаптивная верстка (media queries)

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/geraskinalekseysergeevich/mesto-project-ff
```

2. Перейдите в папку проекта и установите зависимости:

```bash
npm install
```

3. Скопируйте `.env.example` в `.env` и укажите нужные переменные

4. Запустите проект в режиме разработки:

```bash
npm run dev
```

5. После внесения изменений обновите деплой:

```bash
npm run deploy
```
